NoDataFonts

NoDataFonts is a small, growing collection of free-to-use fonts gathered in one place for convenience. It exists mainly so downloading and keeping track of fonts doesn’t turn into a scavenger hunt.

What’s Inside

A variety of fonts organized into folders.

Each folder may include license information if available.

New fonts may be added from time to time.

Why This Repository Exists

It’s simply a personal library made public, so anyone who needs fonts can grab them without digging through dozens of websites.

How to Use

Browse the folders.

Pick a font you like.

Download the file (.ttf, .otf, etc.).

Install it and use it in your projects.

Contributing

If you have free, legally shareable fonts that you think would fit here, feel free to open a pull request.

Support

If you find this repo useful, giving it a ⭐ is appreciated—it helps others discover it too.
